Introdução

Esta é a documentação de código para a biblioteca AwesomeLib. A AwesomeLib é uma biblioteca para trabalhar com dados incríveis.

Esta documentação de código contém informações sobre como usar a biblioteca, como instalar e exemplos de uso.

Instalação

Para instalar a biblioteca, basta baixar o arquivo awesomelib.js e incluí-lo em seu projeto.

<script src="awesomelib.js"></script>

Exemplos de Uso

A seguir estão alguns exemplos de como usar a biblioteca AwesomeLib.

Exemplo 1

const data = [1, 2, 3, 4, 5];
const result = AwesomeLib.sum(data);
console.log(result); // output: 15

Exemplo 2

const data = [
  { name: "João", age: 25 },
  { name: "Maria", age: 30 },
  { name: "Pedro", age: 20 }
];
const result = AwesomeLib.filter(data, person => person.age < 30);
console.log(result); /* output:
[
  { name: "João", age: 25 },
  { name: "Pedro", age: 20 }
] */
